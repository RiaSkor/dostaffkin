# Dostaffkin

**Dostaffkin** — веб-приложение для оформления и отслеживания доставки посылок между странами и внутри стран. Проект демонстрирует интеграцию с Яндекс.Картами, расчёт стоимости доставки и управление заявками через внешнее API.

## Функциональность

- **Главная страница** – краткое описание услуг и преимуществ.
- **Оформление доставки** (`/order`)  
  – Выбор маршрута с автодополнением адресов (Яндекс.Карты).  
  – Выбор размера посылки и скорости доставки.  
  – Расчёт расстояния, времени и итоговой стоимости.  
  – Отправка заявки через API.
- **Отслеживание посылки** (`/track`)  
  – Поиск по номеру отправления.  
  – Отображение маршрута и текущего статуса.

## Технологический стек

- **Языки и фреймворки**: TypeScript, Angular (standalone-компоненты, signals), RxJS
- **Стилизация**: CSS (без препроцессоров)
- **Карты**: Яндекс.Карты API (загрузка через `<script>`)
- **Сетевое взаимодействие**: Angular `HttpClient`
- **Бэкенд**: тестовое API `https://testologia.ru`

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone <url>
   cd dostaffkin
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите сервер разработки:
   ```bash
   ng serve
   ```
4. Откройте приложение в браузере по адресу `http://localhost:4200`.

> **Примечание**: Яндекс.Карты используют встроенные API-ключи. Для работы на локальном сервере дополнительная настройка не требуется.
